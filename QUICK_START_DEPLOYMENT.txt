QUICK DEPLOYMENT STEPS
=============================================================================

1. CREATE GITHUB REPOSITORY
   - Go to https://github.com/new
   - Name: "generic-website"
   - Copy the HTTPS URL

2. PUSH CODE TO GITHUB
   Run in PowerShell (from project directory):
   
   git remote add origin https://github.com/YOUR_USERNAME/generic-website.git
   git branch -M main
   git push -u origin main

3. DEPLOY TO VERCEL
   - Go to https://vercel.com/dashboard
   - Click "New Project"
   - Import your GitHub repository
   - Click "Deploy"

4. CONFIGURE EMAIL (in Vercel Dashboard)
   Settings → Environment Variables:
   
   SMTP_HOST = smtp.gmail.com
   SMTP_PORT = 587
   SMTP_USER = your-email@gmail.com
   SMTP_PASS = your-app-password
   SMTP_FROM = noreply@yourdomain.com
   CONTACT_EMAIL_TO = admin@yourdomain.com

5. REDEPLOY
   - Go to Deployments tab
   - Click redeploy button
   - Wait 2-3 minutes
   - Get your live URL: https://generic-website.vercel.app

6. TEST YOUR SITE
   - Visit your Vercel URL
   - Test all pages load
   - Test contact form sends emails
